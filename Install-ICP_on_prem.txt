Setup new base Ubuntu image on-prem:
  - VM Workstation: 10/11
  - User: ibm/IBMDem0s!    (NOTE: USE THIS PASSWORD!)
  -  CPU: 2 processors x 2 cores = 4
  - RAM: 8192
  - Disk: SCSI - 50G (or more dependent on needs)

1. Configure static IP,  set hostname to mycluster.icp
    a.  /etc/network/interfaces    (See below my example for device ens33)
      iface ens33 inet static
        address 192.168.133.166
        netmask 255.255.255.0
        gateway 192.168.133.2
        dns-nameservers 8.8.8.8  8.8.4.4
    b.  /etc/hosts
         127.0.0.1 localhost
         192.168.133.166   mycluster.icp
    c. set hostname to mycluster.icp in /etc/hostname

    d. reboot or
       systemctl restart networking.service

2. As root runcommand
  sudo bash
  a. Install
    sudo apt update
    sudo apt install curl
    sudo apt -y install openssh-server

  b. Configure
       In file /etc/ssh/sshd_config  Change Port to 2222 in 
       service ssh restart
      
  c. Test ssh setup: 
     ibm$ ssh -p2222 ibm@localhost

3. mkdir ~/script
    curl -L https://raw.githubusercontent.com/shomrai/script/master/wget_ICP_install_scripts|sh
    You get 4 scripts (P1-getICPbits_and_load_images.txt  P2-provision_ubuntu_16.04.txt  P3-setupSSH.txt  P4-installICP_after_provision.txt)

4. Run Part #1 to copy ICP tar.gz filr and load to docker:
   sudo sh P1-getICPbits_and_load_images.txt
